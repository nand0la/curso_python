Sistema para um banco usando orientação a objetos

Algumas dicas:
- Classe "Cliente" tem que herdar de "Pessoa" (Pessoa -> super classe)
    - Classe "Pessoa" tem que ter nome e idade
    - Classe cliente tem que ter conta
    - Conta passada como agregação
        - Quanda a instancia de uma classe é passada para outra em sua inicialização

- Cria classe "ContaCorrente" e "ContaPoupanca" que herdaram da classe "Conta" (Conta -> Super classe)
    - classe "Conta" tem agencia, numero da conta e saldo
    - classe "ContaCorrente" tem limite extra
    - "Conta" deve ter methodo metodo para depositas
    - "Conta" deve ter metodos para sacar abstrato
